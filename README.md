# Local Frontier

Field notes from the frontier of agent systems.

This is a lightweight static scaffold for a publication and workshop focused on local-first agents, MCPs, orchestration, observability, memory, and grounded autonomous workflows.

## Included

- `index.html` - homepage
- `about.html` - manifesto / about page
- `archive.html` - simple archive / thread index
- `404.html` - basic not-found page
- `posts/field-notes-from-the-frontier-of-agent-systems.html` - opening post
- `posts/why-most-agents-are-still-blind-to-the-machines-they-inhabit.html` - second essay
- `posts/umi-shared-machine-reality-for-local-agents.html` - third essay
- `styles.css` - shared visual system

## Things You Might Otherwise Forget

- basic Open Graph and theme metadata on the homepage
- a dedicated `404.html` for static hosting
- an archive page so the site feels like a publication instead of a single essay
- a separate manifesto page so the homepage does not need to carry all the philosophy
